A System for Designers

The client for this design system is actually myself. Specifically, I am using it for my portfolio, but others are more than welcome to use it.

Behind the Name:

As designers, we are paid for our ideas. So I wanted to create a system that conjures up the feeling of planning and ideating. I tried to stick with a blue print-like theme, like using dashes to make something look cuttable. The font has a written marker look while the color scheme is that of a blue print.

Blueprint Design System was created by Riad Nassar. Any similarities to other design systems is purely coincidental.

Riad Nassar is a User Experience Designer. He has an interest in Interaction Design, especially Human-Robot Interaction. He has a background in Psychology and Computer Science. When he's not designing, he likes to Judge local Robotics Competitions.
